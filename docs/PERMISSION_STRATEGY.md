الهدف
----
وضع استراتيجية طلب الأذونات المتوافقة مع Android 10–14 لضمان الحد الأدنى من الأذونات وطلبها عند الحاجة فقط.

قواعد أساسية
----
- عدم طلب أي إذن عند تشغيل التطبيق
- طلب إذن القراءة فقط عند المستخدم يضغط "استيراد ملف"
- الالتزام بـ Scoped Storage وSAF

تفصيل حسب الإصدارات
----
- Android 13+:
  - `READ_MEDIA_IMAGES` عند استيراد صور
  - `READ_MEDIA_VIDEO` عند استيراد فيديو
  - `READ_MEDIA_AUDIO` اختياري
- Android 10–12:
  - `READ_EXTERNAL_STORAGE` فقط عند الحاجة

تدفق طلب الإذن
----
1. المستخدم يضغط "استيراد"
2. عرض شرح modal يوضح لماذا نطلب الإذن (privacy-first)
3. طلب الإذن واستخدام SAF picker
4. إذا رفض، عرض تعليمات بديلة

حالات الحافة
----
- المستخدم رفض الإذن بشكل دائم -> إظهار شاشة توضيحية مع خطوات يدوية

مخاطر
----
- سوء استخدام الأذونات يؤدي لرفض تطبيق Google Play. لاحظ: لا نستخدم MANAGE_EXTERNAL_STORAGE.
