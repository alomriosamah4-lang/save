هدف
----
تحديد نمط التخزين الآمن للملفات وبيانات الميتاداتا الخاصة بالخزائن.

المبادئ
----
- تخزين دائم مشفر فقط داخل Internal App Storage أو داخل صندوق SQLCipher
- لا استخدام Shared/Public folders إلا عند استيراد بواسطة SAF
- لا استخدام MANAGE_EXTERNAL_STORAGE

تفاصيل
----
- قاعدة البيانات: SQLCipher مخفية داخل مسار التطبيق
- الملفات: تُخزن كـ encrypted blobs ضمن بنية chunked داخل storage أو كملفات مشفرة تستخدم XChaCha20-Poly1305
- Thumbnails: مخزنة مشفّرة ومحدودة الحجم، تُنظف تلقائيًا

سياسات الحذف والتنظيف
----
- حذف آمن (overwrite buffer إذا أمكن) عند إزالة ملف
- تنظيف cache دوري (أسبوعي) لفحص orphan files

قيود
----
- عند الاستيراد عبر SAF، نستخدم Streams فقط، ولا ننسخ المحتوى بصيغ نصية على القرص
- الاعتماد على Scoped Storage وSAF لملفات المستخدم
